## How to Build?

1. Get the lastest pull

```sh
git pull
```

2. Install all the dependencies

```sh
yarn
```

3. Now build it.

```sh
yarn build
```

## How to Run Storybook?

1. Take the latest pull

```sh
git pull
```

2. Run the following command -

```sh
yarn storybook
```
